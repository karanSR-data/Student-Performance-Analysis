# 🧠 Student Performance Analysis (Mini Project)

This project is focused on analyzing student performance data to uncover insights using Python libraries like Pandas, Matplotlib, and Seaborn. The analysis helps understand how different factors like gender, parental education, and test preparation impact students' scores in reading, writing, and math.

---

## 📊 Project Workflow

1. **Import Libraries**
   - Loaded essential Python libraries such as `pandas`, `matplotlib.pyplot`, and `seaborn` for data manipulation and visualization.

2. **Load Dataset**
   - Used a CSV file containing student performance data.
   - Displayed the first few rows to get an overview.

3. **Data Exploration**
   - Checked shape, data types, null values.
   - Used `info()` and `describe()` for statistical summary.

4. **Data Cleaning**
   - Renamed columns for easier access.
   - No null values were found, so no imputation required.

5. **Data Analysis & Visualization**
   - Count plots to show gender distribution, parental education, lunch type, and test preparation course.
   - Bar plots to compare mean scores across gender.
   - Scatter plots to observe relationships between math, reading, and writing scores.
   - Heatmap showing correlation between different score types.

6. **Insights**
   - Students who completed test preparation tend to score higher.
   - Parental education level has a visible impact on student performance.
   - Female students generally perform better in reading and writing.
